Le Muret

Open source alumni community management for schools

About the project

Job networking is almost always made in college or after, when one does the same stuff as their peers. We created Le Muret to bring back highschool to networking and solve the diversity equation with people you already know.

Le Muret is a web application made for schools. Using open source technologies and based on the Laravel PHP framework, you can use it for any purpose, personal or commercial.

The systems allows to invite users by batch, when they graduate. Students can register directly or with their Facebook, Twitter, LinkedIn or Google+ accounts. Don’t worry though: school administrations still have to manually accept profiles.

A simple messaging system seamlessly integrates to the profiles, allowing alumnis to get in touch.

Schools don’t have to hire specialists to run Le Muret, as a simple administration interface allows to easily backup, import data and manage users.

DEV CANDY FEATURES

It includes the latest Bootstrap framework, typesafety and multi-device testing along with SASS goodness for easy customization and Gulp/Elixir automation.

Laravel helpers have been customed for an optimal programming workflow allowing you to rapidly modify Le Muret to your needs and hopefully give back to the community as easily ;-)
